Introduction

Hello! This is a demo project made with Spring and Spring Boot.
Please note that it is still work in progress.

After logging in you can switch to "Offices list" tab using the navigation bar at the top of the screen. From there you can click "Make reservation" to reserve a workspace in a selected office. Your reservation should be visible in "Reservations list" tab. You can also see registered users in "Users" tab. If you're logged in from Admin account, you can perform Add/Update/Delete operations on Users, Offices and Reservations.
